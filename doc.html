Get Started

Learn how to use the replication package.

Requirements

Java 8 or later and Maven 3.

Installation

Clone the replication package from Github and build provider and prosumer services.

                     
                       git clone https://github.com/sosygroup/synthesis-context-aware-choreographies.git synthesis-context-aware-choreographies
                       cd synthesis-context-aware-choreographies
                       cd providers
                       mvn clean install
                       cd ../prosumers
                       mvn clean install

Deployment and Enactment

Setup your target nodes

Deploy the war files of the provider and prosumer services on one or more servlet containers (e.g., Apache Tomcat). Be sure that each servlet container is accessibile from the network.
